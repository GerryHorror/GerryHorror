## Selected projects

### [HealingInWriting](https://github.com/ST10046280-Blankenberg/healing-in-writing)
Web platform for a Cape Town non-profit supporting trauma survivors through journaling workshops and events.

- Built with ASP.NET Core MVC and EF Core on .NET 8
- Role-based flows for guests, volunteers, and administrators
- Google Books API integration for ISBN-based resource imports

### [TightBudget](https://github.com/GerryHorror/tightbudget-prog7313)
Android finance app focused on real-world daily budgeting.

- Kotlin + MVVM architecture
- Offline-first sync with Firebase and Room
- Recurring transactions, receipt capture, streaks, and challenges

### [CivicConnect](https://github.com/GerryHorror/civic-connect-prog7312)
Municipal issue reporting and event management system.

- ASP.NET Core MVC + Entity Framework Core
- Custom data structures and algorithms for prioritisation (AVL/BST/heaps/graph traversal)
- Recommendation and status-tracking features for citizen engagement

### [Crypsis](https://github.com/ST10046280-Blankenberg/gerard-st10046280-insy7314-poe)
Secure payments portal built on the MERN stack.

- JWT authentication, RBAC, and step-up authentication
- Security hardening with Helmet, rate limiting, sanitisation, and account lockout workflows
- SonarCloud-backed CI/CD pipeline
